# whitewater-bot

Post Whitewater weather conditions to Mastodon at https://nelson.social/@WhitewaterBot

# Prerequisites

* Node.JS 16+

# Install

```bash
npm install
```

# Run
```bash
MASTODON_TOKEN=<token> MASTODON_API=https://nelson.social/api/ npm run post-status
```
